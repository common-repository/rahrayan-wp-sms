== Frequently Asked Questions ==

= How do I install the SMS plugin? =

To install the plugin, you can use the WordPress auto-update or install it manually.
1. First, use a FTP plugin or WordPress uploader to add the plugin to the wp-content / plugins directory in a separate and special folder. Note that you should not add plugin files outside of your own directory and / wp-content / plugins /.
2. Activate the plugin from the plugin section.
3. To configure the plugin from the Settings section, go to the Settings section of this plugin.
4. Continue to configure your preferred configuration and finally enable plug-in configuration at the top of the configuration section.
5. It's over!
If you plan to manually upgrade the plug-in, you have to do the steps listed step-by-step and then re-enable the plugin.
We emphasize that backup should not be forgotten before updating.
